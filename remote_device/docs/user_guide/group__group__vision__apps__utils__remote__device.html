◆ appRemoteDeviceSendMessage()

int32_t appRemoteDeviceSendMessage ( app_remote_device_channel_t *  channel,
void *  data,
uint32_t  len,
uint32_t  is_response,
uint32_t  request_id,
uint32_t  device_id,
int32_t(*)(void *data, void *msg, uint32_t len)  msg_done_cb,
void *  meta_data 
)

Send a message to remote driver.

Parameters
channel[in] channel (rproc / ept) to send message
data[in] message data
len[in] len of message
is_response[in] whether this is a direct message or a response to a request
request_id[in] id of request if is_response is TRUE
device_id[in] id of virtual device
msg_done_cb[in] callback to free message
meta_data[in] first argument to msg_done_cb
Returns
0 on success, else failure
