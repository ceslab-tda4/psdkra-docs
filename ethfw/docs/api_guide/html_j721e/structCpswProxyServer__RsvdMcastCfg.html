Ethernet Switch Proxy Server APIs

Detailed Description

Ethernet Firmware reserved multicast configuration.

Ethernet Firmware configuration parameters for reserved multicast addresses, that is, multicast addresses that are restricted to Ethernet Firmware only. Requests from remote clients to add any of these multicast addresses to the filter will be rejected.
